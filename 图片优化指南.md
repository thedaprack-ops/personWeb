# 🖼️ 图片优化指南 - WebP 格式转换

## 为什么使用 WebP？

- **更小的文件体积**: 比 JPEG 小 25-35%，比 PNG 小 26-80%
- **更好的压缩质量**: 支持有损和无损压缩
- **现代浏览器支持**: Chrome, Firefox, Edge, Safari 14+
- **2026年标准**: 现代Web开发的推荐格式

## 转换方法

### 方法1: 使用在线工具（推荐）

1. 访问 [Squoosh](https://squoosh.app/)
2. 拖拽你的图片
3. 选择输出格式为 WebP
4. 调整质量（推荐 80-85）
5. 下载转换后的图片

### 方法2: 使用命令行工具

```bash
# macOS (使用 Homebrew)
brew install webp
cwebp -q 80 input.jpg -o output.webp

# Windows (下载)
# 1. 下载 libwebp: https://developers.google.com/speed/webp/download
# 2. 转换图片
cwebp -q 80 input.jpg -o output.webp

# 批量转换 (macOS/Linux)
for file in images/*.jpg; do
    cwebp -q 80 "$file" -o "${file%.jpg}.webp"
done
```

### 方法3: 使用 Python 脚本

```python
from PIL import Image
import os

def convert_to_webp(input_path, output_path, quality=80):
    img = Image.open(input_path)
    img.save(output_path, 'webp', quality=quality)
    print(f'✅ 转换完成: {output_path}')

# 转换所有图片
images_dir = 'images'
for filename in os.listdir(images_dir):
    if filename.endswith(('.jpg', '.jpeg', '.png')):
        input_path = os.path.join(images_dir, filename)
        output_path = os.path.join(images_dir, f"{os.path.splitext(filename)[0]}.webp")
        convert_to_webp(input_path, output_path)
```

## 更新 HTML 引用

转换完成后，在 HTML 中使用 `<picture>` 标签提供回退支持：

```html
<picture>
    <source srcset="images/hero_portrait.webp" type="image/webp">
    <source srcset="images/hero_portrait.jpg" type="image/jpeg">
    <img src="images/hero_portrait.jpg" alt="描述">
</picture>
```
